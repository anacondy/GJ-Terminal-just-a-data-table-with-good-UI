### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/anacondy/GJ-Terminal-just-a-data-table-with-good-UI.git
   cd GJ-Terminal-just-a-data-table-with-good-UI
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Python 3
     python -m http.server 8080
     
     # Python 2
     python -m SimpleHTTPServer 8080
     
     # Node.js (if you have http-server installed)
     npx http-server
     ```

3. **View the site**
   - Navigate to `http://localhost:8080` (or the port your server uses)
